## Atomic Design Organization

This component library follows **Atomic Design** principles, organizing components into five distinct categories:

### 1. Atoms

Basic building blocks that can't be broken down further. These are simple HTML elements or low-level components with a single purpose.

**Buttons:**

- **Button** - Multi-variant button (primary, secondary, ghost, danger)
- **IconButton** - Icon-only button for actions
- **Toggle** - Toggle button with on/off states

**Form Inputs:**

- **Input** - Text input with label and helper text
- **Checkbox** - Checkbox with label
- **Switch** - Toggle switch for boolean values
- **Slider** - Range slider for numeric values
- **RadioGroup** - Group of radio buttons

**Display:**

- **Text** - Typography component with size/weight variants
- **Avatar** - User avatar with image fallback
- **Chip** - Tag/chip component for categorization
- **Alert** - Alert message with severity levels
- **Progress** - Linear progress bar
- **CircularProgress** - Spinner/loader indicator
- **Skeleton** - Loading placeholder

**Layout:**

- **Box** - Core layout wrapper with style props
- **Container** - Width-constrained container
- **Paper** - Surface container with elevation
- **Grid** - CSS Grid layout system
- **AspectRatio** - Enforce aspect ratio for media
- **Spacer** - Flexible spacing utility

**Dividers:**

- **Divider** - Horizontal divider line
- **Separator** - Semantic separator for lists/groups

**Feedback:**

- **Tooltip** - Hover tooltip (Radix UI)
- **Progress** - Progress indicator
- **CircularProgress** - Loading spinner
- **Skeleton** - Loading placeholder

### 2. Molecules

Groups of atoms functioning together as a unit. These components combine one or more atoms to create a more complex UI element.

**Form Components:**

- **FormControl** - Wrapper for form inputs with label and helper text
- **Select** - Dropdown select component

**Lists & Navigation:**

- **List** - Styled unordered list
- **ListItemButton** - Interactive list item
- **Menu** - Dropdown menu (Radix UI)
- **Tabs** - Tab navigation (Radix UI)

**Data Display:**

- **Table** - Styled HTML table
- **DataTable** - TanStack Table wrapper with sorting/pagination

**Layout & Scrolling:**

- **ScrollArea** - Custom scrollable container (Radix UI)
- **Accordion** - Collapsible content sections (Radix UI)

**Interactive Controls:**

- **Dialog** - Modal dialog (Radix UI)
- **Popover** - Popover overlay (Radix UI)
- **Seeker** - Audio/video seek control
- **VolumeSlider** - Volume control slider

### 3. Organisms

Distinct sections of an interface composed of multiple molecules. These are complex components that form discrete sections of the UI.

**Compound Components:**

- **Card** - Card container with header, body, and footer sections
- **Calendar** - Calendar date picker with date selection
- **Toast** - Toast notification with ToastProvider
- **SettingsForm** - Form for managing application settings
