# Music Visualizer Branch

This branch adds a focused music playback experience with visualizers, gapless crossfade, and Sit Back Mode.

## Features

- Web Audio master output with mixer gain and limiter.
- Gapless crossfade between tracks when supported.
- Visualizers: Frequency Analyzer, Waveform, and MilkDrop (Butterchurn).
- Sit Back Mode enhancements on the queue view.
- Settings are persisted server-side per user.

## Where to Configure

Open Playback Settings and find the Music Visualizer section to enable visualizers and tune settings.

## Notes

- Visualizers and crossfade depend on browser Web Audio support.
- This branch targets ES5-compatible browsers.
